# 🚗 MTCars Exploratory Data Analysis

## 📖 Overview

This project presents an Exploratory Data Analysis (EDA) of the famous **Motor Trend Cars (MTCars)** dataset using Python. The analysis focuses on understanding vehicle performance, fuel efficiency, and engine characteristics through statistical summaries and visualizations.

The MTCars dataset contains information on 32 automobile models, including attributes such as miles per gallon (MPG), horsepower, weight, cylinders, transmission type, and more.

---

## 🎯 Project Objectives

* Explore and understand the dataset structure.
* Perform data cleaning and preprocessing.
* Generate descriptive statistics.
* Analyze relationships between variables.
* Visualize trends using charts and graphs.
* Identify factors influencing fuel efficiency and performance.

---

## 📊 Dataset Description

| Column | Description             |
| ------ | ----------------------- |
| mpg    | Miles per Gallon        |
| cyl    | Number of Cylinders     |
| disp   | Engine Displacement     |
| hp     | Gross Horsepower        |
| drat   | Rear Axle Ratio         |
| wt     | Weight (1000 lbs)       |
| qsec   | Quarter-Mile Time       |
| vs     | Engine Type             |
| am     | Transmission Type       |
| gear   | Number of Forward Gears |
| carb   | Number of Carburetors   |

## 🔍 Key Findings

* Vehicle weight negatively impacts fuel efficiency (MPG).
* Manual transmission cars generally achieve better MPG than automatic cars.
* Higher horsepower often correlates with lower fuel efficiency.
* Engine displacement and horsepower show a strong positive relationship.
